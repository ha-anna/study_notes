Primitive types:

  • string
  • number
  • boolean
  • undefined
  • null
  • symbol

Everything else:

  • object type

We can check the type of operator by using typeof:

let message = 'some string';

console.log(typeof message); //logs out string
console.log(typeof 23); // logs out number
console.log(typeof true); // logs out boolean
console.log(typeof window); // logs out object

One type of data can be converted to another:

  • explicit type conversion
    let message = 'some string';
console.log(Boolean(message)); // logs out true
console.log(String(42)); // logs out "42"
  • implicit type conversion (also called ”automatic conversion”)

Implicit Type Conversion

Strings and Numbers

console.log('1' * '2'); // logs out 2; 
// * [multiplication symbol] coerces strings to numbers
// /, %, - will do the same

// HOWEVER + has two functions and this causes confusion
console.log(1 + 1) // logs out 2
console.log("1" + "1") // logs out 11
console.log("1" + 1) // also logs out 11
// string always wins

Booleans: Truthy or Falsy

if (Boolean(value) === true) {
  // if true, do something with value  
}

// what happens here is that JavaScript tries to convert the value of if(), 
//regardless what type it is to a Boolean 
// and then compares that value with true

How to avoid problems when comparing data:

  1. Avoid direct comparisons in conditionals
    //instead of

const username = '';

if (username === false) {
  console.log('no user');
}

//do

if (!username) {
  console.log('no user');
}
  1. Use the triple equals === aka strict equals operator (because loose equals operator == coerces type)
    //instead of

if (null == undefined) {
  console.log('equals');
} else {
  console.log('not equals');
} // logs out equals

// do 

if (null === undefined) {
  console.log('equals');
} else {
  console.log('not equals');
} // logs out not equals
  1. Convert to real Boolean values where needed
    
// instead of

if (NaN === NaN) {
    console.log('equal')
} else {
    console.log('not equals')
}  // logs out not equals


// do

if (Boolean(NaN) === Boolean(NaN)) {
    console.log('equal')
} else {
    console.log('not equals')
} // logs out equals

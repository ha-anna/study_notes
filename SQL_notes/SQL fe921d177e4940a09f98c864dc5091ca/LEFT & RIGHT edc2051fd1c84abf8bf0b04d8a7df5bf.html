LEFT & RIGHT

TypeFunctions
Reviewed

LEFT(string, number_of_chars)
RIGHT(string, number_of_chars)
  • LEFT() function extracts a number of characters from a string (starting from left)
  • RIGHT() function extracts a number of characters from a string (starting from right)

Concatenating

SELECT 
LEFT(first_name, 1) || LEFT(last_name, 1),
first_name, 
last_name
FROM customer

We can concatenate two columns by using ||. This way we can bring characters from two columns together.

SELECT 
LEFT(first_name, 1) || '. ' || LEFT(last_name, 1) || '.',
first_name, 
last_name
FROM customer

We can also add other things in-between.

Example #1

SELECT 
LEFT(first_name, 2),
first_name
FROM customer

Example #2

SELECT 
RIGHT(first_name, 2),
first_name
FROM customer

Example #3

SELECT 
RIGHT(LEFT(first_name, 2), 1),
first_name
FROM customer

One way to extract 2nd letter. There is a better solution though.

Example #4

SELECT 
email,
RIGHT(email, 5),
LEFT(RIGHT(email, 4), 1)
FROM customer

Example #5

SELECT 
email,
LEFT(email, 1) || '***' || RIGHT(email, 19) AS anonymized_email
FROM customer

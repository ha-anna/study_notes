TO_CHAR(date/time/interval, format)
  • used to get custom formats timestamp/date/numbers

Formats

- YYYY-MM-DD -> 2022-12-23
- YYYY-MM -> 2022-12
- Month -> 12 OR DECEMBER
- MM-YYYY -> 12-2022
- YY -> 22
- YYYY -> 2022
- Mon -> 12 OR DEC
- DD -> 23
- DDTH -> 23RD
- DAY -> WEDNESDAY
- HH24 (hours, 24hr clock) -> 20
- HH (hours, 12hr clock) -> 08
- MI (minutes) -> 12
- SS (seconds) -> 33
more here: https://www.postgresql.org/docs/current/functions-formatting.html

Example #1

SELECT
amount,
EXTRACT(month from payment_date) AS payment_month,
TO_CHAR(payment_date, 'MM-YYYY') AS date_format
FROM payment

Example #2

SELECT
SUM(amount),
TO_CHAR(payment_date, 'Day, Month, YY')
FROM payment
GROUP BY TO_CHAR(payment_date, 'Day, Month, YY')

Example #3

SELECT
SUM(amount),
TO_CHAR(payment_date, 'Dy, DD/MM/YYYY')
FROM payment
GROUP BY TO_CHAR(payment_date, 'Dy, DD/MM/YYYY')
ORDER BY SUM(amount)

Example #4

SELECT
SUM(amount),
TO_CHAR(payment_date, 'Month, YYYY')
FROM payment
GROUP BY TO_CHAR(payment_date, 'Month, YYYY')
ORDER BY SUM(amount)

Example #5

SELECT
SUM(amount),
TO_CHAR(payment_date, 'Dy, HH:MI')
FROM payment
GROUP BY TO_CHAR(payment_date, 'Dy, HH:MI')
ORDER BY SUM(amount) DESC

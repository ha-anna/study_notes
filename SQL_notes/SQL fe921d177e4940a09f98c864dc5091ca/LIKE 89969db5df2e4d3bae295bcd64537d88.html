SELECT column_name1, column_name2, ...
FROM table_name
WHERE column_name1 LIKE pattern;
  • LIKE operator is used in a WHERE clause to search for a specified pattern in a column
  • used to filter by matching against a pattern

Wildcards

SELECT *
FROM actor
WHERE first_name LIKE '_A%'

_ - any single character

SELECT *
FROM actor
WHERE first_name LIKE 'A%'

% - any sequence of characters

Example #1

SELECT *
FROM actor
WHERE first_name LIKE '%A%'

Any characters can be between A.

Example #2

SELECT *
FROM actor
WHERE first_name LIKE '__A%'

If there is no character after A% (like in UMA) the result will be included too.

Example #3

SELECT *
FROM actor
WHERE first_name NOT LIKE '%A%'

Shows results that have no letter A in their first_name value.

Example #4 

SELECT *
FROM film
WHERE description LIKE '%Drama%'
AND title LIKE '_T%'

Example #5

SELECT COUNT(*)
FROM film
WHERE description LIKE '%Documentary%'

Example #6

SELECT COUNT(*)
from customer
WHERE first_name LIKE '___'
AND (last_name LIKE '%X'
OR last_name LIKE '%Y')

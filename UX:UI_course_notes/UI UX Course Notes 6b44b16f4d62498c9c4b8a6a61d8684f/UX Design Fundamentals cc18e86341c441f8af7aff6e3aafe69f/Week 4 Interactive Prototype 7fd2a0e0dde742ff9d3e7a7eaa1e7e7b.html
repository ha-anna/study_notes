Feeling Real

We want our digital prototype to feel as real as it possibly can. We want to simulate the finished app as best we can. In order to do that, we need to simulate the platform that the app is going to appear on.

So our digital prototype needs to be specific to the device that it's going to be used on.

We want our user interface design to be polished and believable. And we want the visual and verbal content that we have to be believable as well.
